GPIO

Overview

  • NuMicro NUC123 Series has up to 47 General Purpose I/O pins can be shared with other function pins; it depends on the chip configuration. These 47 pins are arranged in 4 ports named with GPIOA, GPIOB, GPIOC , GPIOD and GPIOF. Each port equips maximum 16 pins. Each one of the 47 pins is independent and has the corresponding register bits to control the pin mode function and data.
