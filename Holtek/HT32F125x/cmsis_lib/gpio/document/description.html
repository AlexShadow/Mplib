GPIO

Overview

    • Up to 32 General Purpose I/O port (GPIO), named PA0 ~ PA15 and PB0 ~ PB15 for the device to implement the functions of input/output. Each of the GPIO ports has the related registers for the control and configuration to make the specific port suitable for the applications.
    • pin-shared with other alternate functions (AFs) to obtain the maximum performance.
    • The external interrupts from GPIO ports of the device have the related control and configuration registers in the External Interrupt Control Unit (EXTI).
    • Input/output direction control.
    • Output push-pull/open drain enable control.
    • Input weak pull-up/pull-down control.
    This component can be used to set General Purpose I/O port mode, configure alternate functions, configure interrupt, etc.

API Reference

 

    GPIO_DeInit

    Deinitializes the GPIOx peripheral registers to their default reset values.
    GPIO_DirectionConfig

    Configures the direction of specified GPIO pins.
    GPIO_PullResistorConfig

    Configures the pull resistor of specified GPIO pins.
    GPIO_InputConfig

    Enables or Disables the input control of specified GPIO pins.
    GPIO_DriveConfig

    Selects the driving current of specified GPIO pins.
    GPIO_OpenDrainConfig

    Enables or Disables the open drain function of specified GPIO pins.
    GPIO_ReadInBit

    Get the input data of specified port pin.
    GPIO_ReadInData

    Get the input data of specified GPIO port.
    GPIO_ReadOutBit

    Get the output data of specified port pin.
    GPIO_ReadOutData

    Get the output data of specified GPIO port.
    GPIO_SetOutBits

    Sets the selected port bits of output data.
    GPIO_ClearOutBits

    Clears the selected port bits of output data.
    GPIO_WriteOutBits

    Sets or Clears the selected port bits of data.
    GPIO_WriteOutData

    Puts data to the specified GPIO data port.
    AFIO_EXTISourceConfig

    Selects the GPIO pin to be used as EXTI channel.
    GPIO_PinLock

    Lock configuration of GPIO pins.
    GPIO_IsPortLocked

    Gets the lock state of specified GPIO port.
    GPIO_IsPinLocked

    Gets the lock state of specified GPIO port pin.
    AFIO_DeInit

    Deinitializes the AFIO peripheral registers to their default reset values.
    AFIO_GPAConfig

    Configures alternated mode of GPIOA with specified pins.
    AFIO_GPBConfig

    Configures alternated mode of GPIOB with specified pins.
