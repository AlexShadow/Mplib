RSTCU

Overview

  • The Reset Control Unit (RSTCU) has three kinds of reset including power on reset, system reset, and
    APB unit reset. Power on reset (Cold reset) resets full system during power up. System reset resets
    the processor core and peripheral IP components but the SW-DP controller are excluded. The resets
    can be triggered by an external signal (off chip), internal events, and reset generators.

API Reference

    RSTCU_GetResetFlagStatus

    Checks the specific global reset flag is set or not.
    RSTCU_ClearResetFlag

    Clear the specific global reset flag.
    RSTCU_APBPerip0Reset

    Enable APB peripheral 0 reset function.
    RSTCU_APBPerip1Reset

    Enable APB peripheral 1 reset function.
    RSTCU_IS_APBPerip0ResetEnd

    This function is used to check APB peripheral 0 reset state.
    RSTCU_IS_APBPerip1ResetEnd

    This function is used to check APB peripheral 1 reset state.
