MISC

Overview

    The misc provide all miscellaneous firmware functions, including nvic and systemtick operations.The nvic functions including Sets the vector table location and offset, Select the condition for the system to enter low power mode, Generate a Core reset. And the SystemTick functions including, Set the pending bit for an system handler, Configures the SysTick clock source, Enable or disables the SysTick counter, Enables or disables the SysTick Interrupt, Sets SysTick counter reload value.

      API Reference

    NVIC_SetVectorTable

    Sets the vector table location and Offset.
    NVIC_LowPowerConfig

    Selects the condition for the system to enter low power mode.
    NVIC_CoreReset

    Generates a Core (Core + NVIC) reset.
    NVIC_SetPendingSystemHandler

    Set the pending bit for an system handler.
    SYSTICK_ClockSourceConfig

    Configures the SysTick clock source.
    SYSTICK_CounterCmd

    Enables or disables the SysTick counter.
    SYSTICK_IntConfig

    Enables or disables the SysTick Interrupt.
    SYSTICK_SetReloadValue

    Sets SysTick counter reload value.
