USART

Overview

  • The Universal Synchronous Asynchronous Receiver Transceiver (USART) provides a flexible full duplex data exchange with synchronous or asynchronous transfer. The USART performs to translate data between parallel and serial interface.The USART module also offers one 16-byte transmitter FIFO (TX_FIFO) and receiver FIFO (RX_FIFO).Both asynchronous and clocked synchronous serial communication modes are supported.aster application.

API Reference

    USART_DeInit

    Deinitializes the USART peripheral registers, with reset USART.
    USART_Init

    Initializes the USARTx peripheral according to the specified parameters in the USART_InitStruct.
    USART_StructInit

    Fills each USART_InitStruct member with its default value.
    USART_SynClockInit

    Initializes the USARTx peripheral Clock according to the specified parameters in the USART_ClockInitStruct.
    USART_SynClockStructInit

    Fills each USART_SynClockInitStruct member with its default value.
    USART_SendData

    USART SendData from Tx.
    USART_ReceiveData

    USART ReceiveData from Rx.
    USART_SetGuardTime

    Sets the specified USART guard time.
    USART_GetLineStatus

    Get LINE Status flags.
    USART_GetModemStatus

    Get Modem status.
    USART_ForceModemPinState

    Force pin DTR/RTS to low or high state.
    USART_IrDACmd

    Enables or disables the USART IrDA interface.
    USART_IrDAConfig

    Configures the USART IrDA interface..
    USART_SetIrDAPrescaler

    Sets the specified USART IrDA prescaler.
    USART_IrDADirectionConfig

    Enables the IrDA transmitter ot receiver.
    USART_RS485TxEnablePolarityConfig

    The RS485 transmitter enable signal polarity configuraion.
    USART_ForceBreakCmd

    Enable or disable forcing Tx pin to logic 0.
    USART_IntConfig

    Enables or disables the USART interrupts.
    USART_GetIntID

    Get Interrupt ID value.
    USART_RFITLConfig

    Rx FIFO Interrupt Trigger Level Configuration.
    USART_TFITLConfig

    Tx FIFO Interrupt Trigger Level Configuration.
    USART_TimeOutIntConfig

    Enables or Disables the USART time out interrupt.
    USART_SetTimeOutValue

    Sets the value of USART FIFO Time Out.
    USART_FIFOReset

    This function is used to clear both the write and read point in Tx FIFO or Rx FIFO.
    USART_StickParityCmd

    Enables or Disables the USART stick parity function.
    USART_StickParityConfig

    Configures the USART stick parity value.
