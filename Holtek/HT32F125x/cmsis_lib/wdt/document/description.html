WDT

Overview

    • 12-bit down counter with 3-bit prescaler
    • Clock source from either internal 32 kHz RC oscillator (LSI) or 32,768 Hz oscillator (LSE)
    • Provides reset or interrupt signals to the system
    • Limited reload window setting to prevent unsuitable Watchdog Timer reload
    • Watchdog Timer may be stopped while the processor is in debug state
    • Restart lock key to prevent unexpected operation
    • Write protection function of configuration register (counter value, interrupt enable, reset enable, delta value, and prescaler)
    This component can be used to set WDT mode (reset or interrupt), set reload and delta value, reload counter (to prevent underflow).

API Reference

    WDT_DeInit

    Deinitializes the WDT peripheral registers to their default reset values.
    WDT_IntConfig

    Enables or Disables the WDT interrupt when WDT meets underflow or error.
    WDT_ResetCmd

    Enables or Disables the WDT Reset when WDT meets underflow or error.
    WDT_ProtectCmd

    Enables or Disables protect mechanism of the WDT.
    WDT_SetReloadValue

    Sets Reload value of the WDT.
    WDT_GetReloadValue

    Get the current reload value of the WDT.
    WDT_SetDeltaValue

    Sets delta value of the WDT.
    WDT_GetDeltaValue

    Get the current delta value of the WDT.
    WDT_SetPrescaler

    Sets Prescaler value of the WDT.
    WDT_GetPrescaler

    Get the current prescaler value of the WDT.
    WDT_Restart

    WDT Restart (Reload WDT Counter).
    WDT_GetFlagStatus

    Check whether the specified WDT flag has been set.
