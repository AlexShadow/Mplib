RTC

Overview

    • Real Time Clock (RTC) includes APB interface, 32 bits up counter, control register, prescaler, compare register, and status register. Most of the RTC portion located in Backup Domain except the APB interface. RTC is used as a wakeup timer to let the system resume from Power-Down mode.
    • 32 bits up counter for counting elapsed time
    • Three kinds of Interrupt/wakeup setting: RTC second clock interrupt/wakeup 、RTC match interrupt/wakeup and RTC counter overflow interrupt/wakeup
    • RTC clock source could be any of the following :LSE oscillator clock 、LSI oscillator clock
    • Programmable clock prescaler.Division factor: 1, 2, 4, 8…,32768
    This component can be used to set RTC time, configure Alarm function, configure interrupt, read and write backup registers.

API Reference

    RTC_DeInit

    Deinitializes the RTC peripheral registers to their default reset values.
    RTC_ClockSourceConfig

    Selection of RTC timer clock source.
    RTC_LSILoadTrimData

    Loads the LSI trim data.
    RTC_LSICmd

    Enables or Disables LSI clock.
    RTC_LSESMConfig

    Selects LSE startup mode.
    RTC_LSECmd

    Enables or Disables LSE clock.
    RTC_CMPCLRCmd

    Enables or Disables RTC counter cleared by compare match function.
    RTC_SetPrescaler

    Config RTC prescaler to a specified value.
    RTC_GetPrescaler

    Returns the value of divider.
    RTC_Cmd

    Enables or Disables RTC timer.
    RTC_GetCounter

    Returns the counter value of RTC timer.
    RTC_SetCompare

    Configures the compare match value.
    RTC_GetCompare

    Returns the compare match value of RTC timer.
    RTC_WakeupConfig

    Enables or Disables the specified wakeup source.
    RTC_IntConfig

    Enable or Disables the specified interrupt source.
    RTC_GetFlagStatus

    Returns the RTC flags.
    RTC_OutConfig

    Configures the RTC output function.
    RTC_OutCmd

    Enables or Disables RTC output.
    RTC_GetOutStatus

    Returns the RTCOUT level mode flag.
