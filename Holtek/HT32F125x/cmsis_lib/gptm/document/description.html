GPTM

Overview

  • The General-Purpose Timer (GPTM0 and GPTM1) is consisted of one 16-bit up/down-counter, four 16-bit CCRs (Capture/Compare Register), one 16-bit counter-reload counter and several control/status registers. It can be used for a variety of purposes; including measurement of the pulse width of the input signal or generation of the output waveforms (compare match output, PWM output). The GPTM supports Encoder Interface by a decoder with two inputs. This GPTM can also support motor control; Hall sensor interface.
