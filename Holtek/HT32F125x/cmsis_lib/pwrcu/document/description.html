PWRCU

Overview

    Power Control Unit (PWRCU) provides many kinds of power saving mode such as Sleep, Deep-Sleep1, Deep-Sleep2, and Power-Down mode to reduce the power consumption.There are three digital power domains (backup domain is excluded) can be turned off properly under some conditions.Three different power domains are backup domain, 3.3V power domain, and 1.8V power domain. Both of internal voltage regulator and depletion MOS can supplie 1.8V power.Battery power (VBAT) will be used for backup domain when VDD33 is shut down.
    This component can be used to read/write specified backup register, enters SLEEP mode, enters POWER-DOWN Mode, configures LVD, BOD, DMOS, external WAKEUP pin function, etc.

API Reference

    PWRCU_DeInit

    Deinitializes backup domain which contains PWRCU and RTC units.
    PWRCU_CheckReadyAccessed

    Waits, until the RTC & PWRCU can be accessed.
    PWRCU_GetFlagStatus

    Returns the flags of PWRCU.
    PWRCU_ReadBackupRegister

    Returns the value of specified backup register.
    PWRCU_WriteBackupRegister

    Writes the DATA to specified backup register.
    PWRCU_Sleep

    Enters SLEEP mode.
    PWRCU_DeepSleep1

    Enters DEEP-SLEEP Mode 1.
    PWRCU_DeepSleep2

    Enters DEEP-SLEEP Mode 2.
    PWRCU_PowerDown

    Enters POWER-DOWN Mode.
    PWRCU_SetLVDS

    Configures LVD voltage level.
    PWRCU_LVDCmd

    Enables or Disables LVD function.
    PWRCU_BODCmd

    Enables or Disables BOD reset function.
    PWRCU_BODRISConfig

    Select when the BOD occurs, the action for the cause Reset or Interrupt.
    PWRCU_GetLVDFlagStatus

    Return the flag status of LVD .
    PWRCU_GetBODFlagStatus

    Return the flag status of BOD .
    PWRCU_GetDMOSStatus

    Returns the DMOS status.
    PWRCU_DMOSCmd

    Enables or Disables DMOS function.
    PWRCU_V18RDYSourceConfig

    Configures VDD18 power good source.
    PWRCU_LVDWakeupIntConfig

    Enables or Disables LVD wakeup and interrupt function.
    PWRCU_WakeupPinCmd

    Enables or Disables external WAKEUP pin function.
    PWRCU_WakeupPinIntConfig

    Enables or Disables external WAKEUP pin interrupt function.
    PWRCU_HSIReadyCounterBitLengthConfig

    Configures HSI ready counter bit length.
