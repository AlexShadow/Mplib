CKCU

Overview

    The Clock Control unit (CKCU) provides functions of high speed internal RC oscillator (HSI), high speed external crystal oscillator (HSE), low speed internal RC oscillator (LSI), low speed external crystal oscillator (LSE), Phase Lock Loop(PLL), HSE clock monitor,clock prescaler, clock multiplexer and clock gating.
    Features :
    • 4 ~ 16 MHz external crystal oscillator (HSE)
    • Internal 8 MHz RC oscillator (HSI) with configuration option calibration
    • PLL with selectable clock source (from HSE or HSI) for system clock
    • 32,768 Hz external crystal oscillator (LSE) for Watchdog Timer or RTC
    • Internal 32 kHz RC oscillator (LSI) for Watchdog Timer or RTC
    • HSE clock monitor
    This component can be used to configure HSE, HSI, PLL, SYS_CK, HCLK, USART divider, ADC clock divider, WDT clock source and APB peripheral clock, return the ready flag, etc.

API Reference

    CKCU_DeInit

    Deinitializes the CKCU registers to the reset values.
    CKCU_HSECmd

    Enables or disables the external high speed oscillator (HSE).
    CKCU_HSICmd

    Enables or disables the internal high speed oscillator (HSI).
    CKCU_PLLCmd

    Enables or disables the PLL clock.
    CKCU_WaitHSEReady

    Waits for HSE ready to use.
    CKCU_GetClockReadyStatus

    Checks whether the specific clock ready flag is set or not.
    CKCU_PLLInit

    This function is used to configure PLL.
    CKCU_PLLSourceConfig

    Configures the PLL clock source.
    CKCU_SysClockConfig

    Configures the system clock (SYS_CK).
    CKCU_GetSysClockSource

    Returns the source clock which is used as system clock.
    CKCU_SetHCLKPrescaler

    Configures the AHB clock (HCLK).
    CKCU_SetUSARTPrescaler

    Configures the USART divider.
    CKCU_SetADCDivide

    Configures the ADC clock divider.
    CKCU_WDTSourceConfig

    Configures the WDT clock source (WDT_CK).
    CKCU_GetClocksFrequency

    Returns the frequency of the different clocks.
    CKCU_APBPerip0ClockConfig

    Enables or disables the APB peripheral 0 clock.
    CKCU_APBPerip1ClockConfig

    Enables or disables the APB peripheral 1 clock.
    CKCU_CKMCmd

    Enables or disables the HSE Clock Monitor function.
    CKCU_PSRCWKUPCmd

    Enables or disables the power saving wakeup RC clock.
    CKCU_CKOUTConfig

    Selects the output clock source through the CKOUT pin.
    CKCU_GetIntStatus

    Checks whether the specific CKCU interrupt has occurred or not.
    CKCU_ClearIntFlag

    Clear the CKCU interrupt flag.
    CKCU_IntConfig

    Enables or disables the specific CKCU interrupts.
    CKCU_SleepClockConfig

    Enables or disables SRAM/FMC clock during Sleep mode.
    CKCU_IS_PLL_USED

    Check PLL clock used by the specific target source or not.
    CKCU_IS_HSI_USED

    Check HSI clock used by the specific target source or not.
    CKCU_IS_HSE_USED

    Check HSE clock used by the specific target source or not.
    CKCU_MCUDBGConfig

    Enables or disables the specific debug function.
    CKCU_BKISOCmd

    Enables or disables the Backup domain isolation control.
