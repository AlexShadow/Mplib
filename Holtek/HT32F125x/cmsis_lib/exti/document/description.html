EXTI

Overview

    The External Interrupt / Event Controller (EXTI) is comprised of 16 edge detectors which can generate event/ interrupts requests independently. In interrupt mode, five source trigger types can be selected(low level, high level, negative edge, positiveedge, and both edge) by setting the EXTICFGRn register.In event mode, wake up event polarity can be configured by setting the EXTIWAKUPPOLR register. If the EVWUPIEN bit in the EXTIWAKUP Register is set, EVWUP interrupt can be generated when the event occurs. Each EXTI line can also be masked independently.

API Reference

 

    EXTI_DeInit

    Deinitializes the EXTI peripheral registers.
    EXTI_Init

    Initializes the EXTI peripheral.
    EXTI_IntConfig

    Enables or Disables the specified EXTI channelx interrupts.
    EXTI_WakeupEventConfig

    Configures the EXTI channelx event wakeup function.
    EXTI_WakeupEventIntConfig

    Enables or Disables the EXTI channelx event wakeup interrupt.
    EXTI_ClearEdgeFlag

    Clears the specified EXTI channelx edge flag.
    EXTI_ClearWakeupFlag

    Clears the specified EXTI channelx wakeup flag.
    EXTI_GetWakeupFlagStatus

    Gets the specified EXTI channelx wakeup flag.
    EXTI_SWIntCmd

    Activates or Deactivates an EXTI channelx interrupt by software.
    EXTI_GetSWCmdStatus

    Gets the specified EXTI channelx software command register bit.
