CMP_OP

Overview

  • Two Operational Amplifiers/Comparators (OPA/CMP) are implemented within the chip. They could be configured to be OPA or analog comparator. When configured as a comparator, it is capable of asserting interrupts to NVIC.

API Reference

    CMP_OP_DeInit

    Deinitializes the CMP_OP peripheral registers to their default reset values.
    CMP_OP_Init

    Initializes the CMP_OPx peripheral according to the specified parameters in the CMP_OP_InitStruct.
    CMP_OP_Cancellation

    Cancellation of CMP_OPx peripheral that will decrease the offset between CP and CN.
    CMP_OP_IntConfig

    Enables or Disables the specified CMP_OP interrupts.
    CMP_OP_GetIntStatus

    Checks whether the specified ADC interrupt has occurred.
    CMP_OP_GetFlagStatus

    Checks whether the specified CMP_OP flag has been set.
    CMP_OP_ClearIntPendingBit

    Clears the CMP_OPx interrupt pending bits.
