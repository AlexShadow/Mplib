FMC

Overview

  • The Flash Memory Controller (FMC) provides functions of flash operation and pre-fetch buffer for the embedded on-chip Flash memory. Figure below shows the block diagram of FMC which includes programming interface, control register, pre-fetch buffer, and access interface. Since the access speed of Flash memory is slower than the CPU, a wide access interface with pre-fetch buffer is provided to the Flash memory in order to reduce the wait state (which will cause instruction gaps) of the CPU. The functions of word program/page erase are also provided for instruction/data storage of Flash memory

API Reference

 

    FLASH_SetWaitState

    Configures the FLASH wait state.
    FLASH_SetPowerSaving

    Enables or Disables FLASH power saving mode.
    FLASH_SetPrefetchBuffer

    Enables or Disables FLASH pre-fetch buffer.
    FLASH_SetRemappingMode

    Sets vector remapping mode.
    FLASH_ErasePage

    Erases a specific FLASH page.
    FLASH_EraseOptionByte

    Erases FLASH Option Byte page.
    FLASH_MassErase

    Erases the entire FLASH.
    FLASH_ProgramWordData

    Programs one word data.
    FLASH_EnablePageWriteProtection

    Enables write protection for the specific pages.
    FLASH_EnableSecurity

    Enables security lock for the page of main FLASH or option byte.
    FLASH_GetWriteProtectionStatus

    Returns write-protection status of FLASH page .
    FLASH_GetSecurityStatus

    Returns security status of the FLASH.
    FLASH_IntConfig

    Enables or Disables the specific FLASH interrupts.
    FLASH_GetIntStatus

    Returns flag status of the FLASH interrupt.
    FLASH_ClearIntFlag

    Clear specific interrupt flags of FLASH.
    FLASH_WaitForOperationEnd

    Waits till the FLASH operation has finished or time-out has occurred.
