I2C

Overview

  • The I2C interface is connected to I2C bus by two serial signals, which is known as a serial data line (SDA) and a serial clock line (SCL). The I2C module provides two types of data transfer rate: (1) 100 kHz in Standard-mode or (2) 400 kHz in Fast-mode. By defining the SCL period generation register, different kinds of duty cycle can be reached for SCL pulse. The SDA line which is connected with the whole I2C bus is bi-directional between master and slave device to transmit or receive data. The I2C module also has the function of detecting arbitration to prevent more than one master from trying to transmit.

API Reference

    I2C_DeInit

    Deinitializes the I2C peripheral registers to their default reset values.
    I2C_Init

    Initializes the I2Cx peripheral according to the specified parameters in the I2C_InitStruct.
    I2C_StructInit

    Fills each I2C_InitStruct member with its default value.
    I2C_Cmd

    Enables or Disables the specified I2C peripheral.
    I2C_GenerateSTOP

    Generates STOP condition of I2C communication.
    I2C_IntConfig

    Enables or Disables I2C interrupts.
    I2C_GeneralCallCmd

    Enables or Disables I2C General Call.
    I2C_AckCmd

    Enables or Disables I2C sending acknowledgment.
    I2C_SetOwnAddress

    Configures own address of the specified I2C.
    I2C_TargetAddressConfig

    Starts transmitting to target slave address.
    I2C_SendData

    Sends a data word through the I2Cx peripheral.
    I2C_ReceiveData

    Returns the received data by the I2Cx peripheral.
    I2C_ReadRegister

    Reads the specified I2C register and returns its value.
    I2C_GetFlagStatus

    Check whether the specified I2C flag has been set.
    I2C_CheckStatus

    Check whether the specified I2C status has been active .
    I2C_ClearFlag

    Clear the specified I2C flag.
    I2C_SetSCLHighPeriod

    Sets the interval timing of the high period of the I2C clock.
    I2C_SetSCLLowPeriod

    Sets the interval timing of low period of the I2C clock.
