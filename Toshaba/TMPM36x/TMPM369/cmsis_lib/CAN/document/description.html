Overview

  • Compliant with CAN version 2.0 B (active)Standard and extended formats supported;Data frames and remote frames supported for each format;32 Mailboxes (31 receive and transmit, 1 receive only);CAN bus baud rate up to 1 Mbps (with a system clock of at least 48 MHz);Bit timing parameter equivalent to Intel 82527; ;Baud rate prescaler built in;The order in which messages are transmitted can be selected from the following two types of internal arbitrations:- The mailbox with the lower number will be sent first,- The mailbox with the higher priority identifier will be sent first;Time stamp function for receive and transmit messages
