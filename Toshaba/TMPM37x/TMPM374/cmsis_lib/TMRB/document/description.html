TRMB

Overview

TRMB have the eight channels multi-functional 16-bit timer/event counter. (TMRB0 through TMRB7) TMRBs operate in the following four operation modes:

  • 16-bit interval timer mode
  • 16-bit event counter mode
  • 16-bit programmable square-wave output mode (PPG)
  • External trigger programmable square-wave output mode (PPG)
The use of the capture function allows TMRBs to perform the following two measurements Pulse width measurement
One-shot pulse generation from an external trigger pulse
